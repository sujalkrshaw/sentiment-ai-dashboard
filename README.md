# 📊 Social Media Sentiment Analysis Dashboard

> A full-stack Machine Learning + NLP project that analyzes social media text and classifies sentiment in real time using an interactive dashboard.

---

## 🚀 Project Overview

The **Social Media Sentiment Analysis Dashboard** is an end-to-end data science application that processes user-generated content (tweets, comments, reviews) and classifies it into:

* 🟢 Positive
* 🔴 Negative
* ⚪ Neutral

It provides actionable insights through a **Streamlit-based interactive dashboard**, enabling businesses to monitor public opinion, customer satisfaction, and brand perception.

---

## 🎯 Problem Statement

Organizations receive massive volumes of textual feedback across social media platforms.
Manual analysis is:

* ❌ Time-consuming
* ❌ Error-prone
* ❌ Not scalable

This project automates sentiment detection using **Natural Language Processing (NLP)** and **Machine Learning**, making analysis fast, consistent, and scalable.

## ▶️ Installation & Setup

### 1️⃣ Clone Repository

```bash
git clone https://github.com/sujalkrshaw/sentiment-ai-dashboard.git
cd sentiment-ai-dashboard
```

---

### 2️⃣ Create Virtual Environment

```bash
python -m venv venv
venv\Scripts\activate
```

---

### 3️⃣ Install Dependencies

```bash
pip install -r requirements.txt
```

---

## ▶️ Run the Application

```bash
streamlit run app/app.py
```

👉 Open in browser:

```
http://localhost:8501
```
